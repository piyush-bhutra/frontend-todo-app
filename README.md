# Frontend To-Do List App

A simple to-do list web application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Remove tasks by clicking on them
- Clean and minimal user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in any web browser

## Purpose
This project was built to practice frontend fundamentals including
DOM manipulation, event handling, and basic styling.
